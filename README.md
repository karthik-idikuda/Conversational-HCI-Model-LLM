# Conversational HCI Model LLM

Research-grade evaluation pipeline for conversational human-computer interaction models, with multi-format reporting and LLM comparison benchmarks.

## Features

- Multi-LLM evaluation (GPT, Claude, Gemini)
- Automated conversation quality scoring
- Response latency and coherence metrics
- Flask-based dashboard for result visualization
- Export to PDF, CSV, and JSON

## Tech Stack

Python, Flask, OpenAI API, Scikit-learn, Pandas

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/Conversational-HCI-Model-LLM.git
cd Conversational-HCI-Model-LLM
pip install -r requirements.txt
```

### Usage

```bash
python evaluate.py
```
